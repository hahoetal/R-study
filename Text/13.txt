통계 분석 기법을 이용한 가설 검증

★ 통계 분석
- 기술 통계: 데이터를 요약해 설명하는 통계 기법
- 추론 통계: 데이터 요약 + 어떤 값이 발생할 확률 추론

- 통계적 가설 검증: 유의확률을 이용해 가설을 검증하는 방법.

* 통계 용어는 기울여서 쓴다!!!
- 유의확률(Significance probability, p-value): 실제로는 집단 간 차이가 없는 우연히 차이가 있는 데이터가 추출될 확률.
				        귀무가설이 사실일 확률...
				        p-value > α : 통계적으로 유의하지 않음, 귀무가설 채택.
				        p-value < α : 통계적으로 유의, 대립가설 채택.

* 유의수준(α): 귀무가설이 참인데도 불구하고 이를 기각하는 확률, 1종오류를 범할 확률의 최대 허용치, 보통 .05 아니면 .01
* 연구가설: 연구문제에 대한 잠정적인 답, 일반적으로 독립변수과 종속변수로 구성.
* 귀무가설(영가설, H0): 기각할 것을 염두에 두는 가설, 연구가설과 논리적으로 반대의 입장을 취하는 가설, 검증을 거쳐야 하는 가설. 
* 대립가설(H₁): 영가설에 대립되는 가설로 영가설이 거짓일 때 채택하기 위해 설정하는 가설, 연구자가 주장하고자 하는 가설로 종종 연구가설과 동일시.

★ t 검정
- 두 집단의 평균비교.
- 독립변수는 한 개.
- 집단의 분산이 같을 때와 다를 때 사용하는 방법이 다름.
- t(df) = t-value
- p = p-value 아니면 p < .001

- t.test(data =   , 비교 변수 ~ 집단 변수, var.equal = T) # 집단의 분산이 동일
- t.test(data =   , DV ~ IV, var.equal = F) # 집단의 분산이 동일하지 않음... 아마도?

★ 상관분석
- 두 연속 변수가 서로 관련이 있는지 검정하는 통계 분석법.
- 상관계수(Correlation Coefficient): 두 변수의 관계 크기와 방향성을 알려 줌, 0과 1 사이의 값을 가짐, 1에 가까울수록 관계성이 큼, 양수면 정비례, 음수면 반비례의 관계를 가짐.

- cor.test(비교할 연속변수1, 비교할 연속변수2)

★ 상관행렬(Correlation matrix): 여러 변수의 관련성을 한 번에 알아보려고 하는 경우 사용.
- cor(데이터)
- 숫자가 너무 많아서 알아보기 힘들어서 상관행렬 히트맵을 만들기도 함.

★ 상관행렬 히트맵(값의 크기를 색깔로 표현한 그래프) 만들기
1) install.packages("corrplot")
2) library(corrplot)
3) corrplot(데이터)
* 상관계수가 클수록 원이 크고, 색깔이 진하게 그려짐. 한 눈에 파악하기는 쉽지만... 정확한 상관계수를 알 수 없다.

★ corrplot 파라미터를 이용해 그래프의 형태 다양하게 바꾸기
	예시)
	      corrplot(car_cor, method = "number") # 원이 아닌 숫자로 표현.
	      
	      col <- colorRampPalette(c("#bb4444", "#ee9988", "ffffff", "#77aadd", "#4477aa")) # 색상 코드 목록을 생성하는 col() 함수 생성.
	      corrplot(car_cor, # 상관행렬
		    method = "color",         # 색깔로 표현
		    col = col(200),              # 색상 200개 선정
		    type = "lower",             # 왼쪽 아래 행렬만 표시
		    order = "hclust",           # 유사한 상관계수끼리 군집화
		    addCoef.col = "black",    # 상관계수 색깔
		    tl.col = "black",             # 변수명 색깔
		    tl.srt = 45,                   # 변수명 45도 기울임
		    diag = F)                    # 대각 행렬 제외
