'dplyr' package
: 데이터 처리 패키지.
install.packages('dplyr') 또는 library(dplyr), 즉 설치 또는 패키지를 불러온 다음에 사용 가능

*파이프 연산자('%>%')
: dplyr 패키지에서 '그리고'와 같은 의미로 이해...
기본 형식) 데이터 프레임 %>% 함수1 >%> 함수2 ... 함수N

* parameter(인자) vs argument(인수)
: parameter(인자)는 함수 내의 변수를 말하고, argument(인수)는 해당 함수에 실질적으로 넣어주는 값.

함수들~
- head(가지고 온 데이터)
: 외부에서 데이터를 불러온 경우, 데이터를 잘 가져왔는지 확인하는 함수로 표 형식으로 보여줌.

- mutate(새로운 열의 이름 = 열을 구하는 방법)
: 가지고 온 데이터에 새로운 열을 추가하고 싶을 때 사용하는 함수.
한 번에 여러개의 열을 생성할 수 있으며, 이때는 ','로 서로 구분.

- transmute(새로운 열의 이름 = 열을 구하는 방법)
: 새로운 열로 구성된 데이터 프레임을 반환하고 기존 데이터 프레임은 삭제...

- raname(새로운 열 이름 = 바꿔줄 열 이름)
: 열 이름을 수정하고 새로운 데이터 프레임을 반환.
names() 또는 colnames() 함수를 통해 이름이 제대로 변경되었는지 확인 가능.
해당 함수들은 인수(argument)로 데이터프레임을 받고 열 이름 포함한 백터를 반환.

- arrange(열 이름)
: 순서대로 정렬, 오름차순

- arrange(desc(열 이름))
: 내림차순

- bind_rows()
: 각각의 데이터 프레임을 하나로 묶기.

- select()
: 원하는 열들로 구성된 새로운 데이터 프레임 생성.
	예시) df %>% select(-id) # id를 제외한 새로운 데이터 프레임을 생성.
